# ChroLens Project

ChroLens 系列自動化工具的官方網站。

## 網站地址

https://lucienwooo.github.io/ChroLens-Project/

## 本地開發

```bash
npm install
npm run dev
```

## 構建

```bash
npm run build
```

構建後的檔案會輸出到 `out` 目錄。

## 部署

推送到 `main` 分支後，GitHub Actions 會自動構建並部署到 GitHub Pages。
